📜 Auto Reels Scroller (Firefox Extension)

A lightweight Firefox extension that automatically scrolls Instagram Reels when videos finish and skips ads—giving you a smooth, uninterrupted browsing experience.

✨ Features
▶️ Auto-scroll on video end
Automatically advances to the next Reel when the current one finishes.
🚫 Ad skipping
Detects and scrolls past sponsored/advertisement content.
⚡ Lightweight & fast
Minimal overhead with no unnecessary tracking or bloat.
🔒 Privacy-friendly
No data collection, no external requests—everything runs locally.
